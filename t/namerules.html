Name guidelines

Please don't make references to rape.

Please don't add ornament to your name to make it stand out. For example, "XxX~Kawaii~Robot~XxX" can be trimmed to "Kawaii Robot."

Please don't write your name in all caps. Exception: when it's an acronym.

Please don't name yourself "test," because test robots shouldn't be submitted.

Profanity is ok, but please leave out racial or gender slurs.

Thank you!
