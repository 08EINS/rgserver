FAQ

What version of Python is the server using?

CPython 2.7.3.

What modules are available?

The entire Python standard library.

How do you make users don't destroy your server?

Each robot's code is run:

  • in a separate process
  • in a chroot jail
  • as user "nobody"
  • firewalled by iptables
  • restricted by ulimit from forking or using too much memory
